- 每个 solution 都包含单元测试

# 脚本使用说明(c/test.sh)
``` sh
# 执行第1个问题
c/test.sh 1

# 执行全部问题
c/test.sh all

# 执行库文件的测试
c/test.sh test

# -m 开启内存泄露检查, 使用valgrind, 如
c/test.sh -m 1
c/test.sh -m all
c/test.sh -m test
```
